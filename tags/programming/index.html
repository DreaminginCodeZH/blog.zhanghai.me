由来

语法规则和特定技巧很大程度上只是记忆的问题，而在几个事实上知晓与否并不能断定面试者的高下。C 语言是一门十分接近底层实现的语言，许多的设计决定与实现直接相关，如果想要合适地运用，也要求程序员对于 C 的实现有透彻的理解。因此，这几个问题主要考察的是面试者在 C 原理与设计层面上的理解。
